# Usage
<!-- usage -->
```sh-session
$ npm install -g pith
$ pith COMMAND

$ pith (-v|--version|version)

$ pith --help [COMMAND]
USAGE
  $ pith COMMAND
...
```

## `pith hello [FILE]`

describe the command here

```
USAGE
  $ pith hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ pith hello
  hello world from ./src/hello.ts!
```

_See code: [src/commands/hello.ts](https://github.com/ashutoshpith/pith/blob/v1.0.0/src/commands/hello.ts)_

## `pith help [COMMAND]`

display help for pith

```
USAGE
  $ pith help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.2.3/src/commands/help.ts)_
